# Number Guessing Game (Python)

This is a beginner-friendly Python project where the computer randomly selects a number between 1 and 100, and the user tries to guess it.

## Features
- Random number generation
- Limited attempts
- Input validation
- Score system
- Play again option

## Technologies Used
- Python
- random module

## How to Run
1. Make sure Python is installed
2. Run the program using:
```bash
python main.py
